Для выполнения некоторых заданий потребуется отключить ASLR. На Linux это можно сделать командой

echo 0 | sudo tee /proc/sys/kernel/randomize_va_space

Флаги gcc, разрешающие исполнение кода на стеке и отключающие "канарейки", есть в CMakeLists.txt. Если стек по каким-то причинам все еще защищен, снать защиту можно утилитой execstack

Для сборки под Windows используйте MinGW или переключитесь на WSL.
